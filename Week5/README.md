
# Week 5 - Text Generation using RNN, LSTM, and GRU

## Objective

Build and compare **Vanilla RNN, LSTM, and GRU** models for text generation using a custom text corpus and analyze their performance.

---

## Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib

---

## Models Implemented

* Vanilla RNN
* LSTM
* GRU

---

## Improvements Made

* Used a custom text corpus.
* Increased embedding dimension from **32 → 64**.
* Increased hidden units from **64 → 128**.
* Increased training epochs from **100 → 200**.
* Generated **10 words** instead of **5**.

---

## Results

* Compared training loss of all three models.
* Evaluated the generated text for each architecture.
* Observed that **LSTM** and **GRU** produced more meaningful sequences than the Vanilla RNN.

---

## Conclusion

This project demonstrates how different recurrent neural network architectures learn sequential patterns and generate text. It also highlights the impact of hyperparameter tuning on model performance and text quality.
