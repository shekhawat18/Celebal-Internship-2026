
# Week 8 Assignment – Single Agent Pipeline

## Overview
This project implements a simple AI Agent that routes user queries to the appropriate tool using conditional logic.

## Features
- Calculator Tool
- Keyword Extraction Tool
- Word Counter Tool (Bonus)
- Conditional Routing
- Basic Error Handling
- Logging for Tool Selection
- Interactive Mode

## Technologies Used
- Python

## Sample Queries
- `Calculate 20 + 5`
- `20 * 15`
- `Extract keywords from Artificial Intelligence is transforming industries`
- `Count words in Machine Learning is amazing`
- `What is machine learning?`

## Learning Outcomes
- Understanding AI Agent architecture
- Implementing tool routing using conditional logic
- Integrating multiple tools into a single agent
- Handling invalid inputs gracefully
- Building an interactive command-line agent
